# Akorfa Fixit Frontend

React + Vite frontend for Akorfa Fixit application.

## Quick Start

```bash
npm install
npm run dev
```

App runs on port 5000.

## Environment Variables

```
VITE_API_URL=https://your-backend-url.com/api
```
